### 2. Feature Requests
- Open an Issue to discuss new ideas before implementation.
- We are particularly interested in:
  - Better Vulkan fluid simulation shaders.
  - Rust-based secure memory management patterns.
  - Mojo-based local AI models.

### 3. Code Contributions
- Fork the repository.
- Create a feature branch (`git checkout -b feature/cool-new-fluid-effect`).
- Commit your changes with clear messages.
- Push to the branch and open a Pull Request.
